About Ramiro

I’m Ramiro Alvarez-Cruz, a Computer Science student at UNC Charlotte who enjoys building clean, accessible web experiences and solid back-end logic. My toolkit spans Java, C/C++, C#, Python, HTML/CSS/JS, and I’m comfortable with data structures, recursion, memory management, file I/O, and RISC-V.

Recent work includes a hash-map document search engine (WordFrequency/HashDocument), a recursive maze solver with coin collection and JUnit tests, a queue-based call-center simulator, and a relational database for a fashion brand’s online catalog (ER → SQL). I’m especially interested in the intersection of creative tech and fashion—the kind of interactive work you see from studios like UNIT9, FutureDeluxe, or BUCK.
